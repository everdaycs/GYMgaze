# 机器人移动方向显示

## 修改说明

将机器人的箭头显示从**静态朝向**改为**动态移动方向**指示器。

## 视觉反馈

### 2D世界视图
- **前进时**：红色粗箭头（指向移动方向，长度随速度变化）
- **后退时**：橙色粗箭头（指向后退方向）
- **静止时**：淡蓝色细箭头（显示朝向）

### 占用栅格地图
- **前进时**：红色箭头
- **后退时**：橙色箭头  
- **静止时**：淡黄色细箭头

## 技术实现

### 运动模型
机器人使用**差动驱动模型**：
- `velocity`：线速度（正=前进，负=后退）
- `angular_velocity`：角速度（旋转）
- 移动方向 = 朝向方向（差动驱动特性）

### 代码逻辑
```python
if abs(velocity) > 0.01:  # 运动中
    if velocity > 0:
        # 前进：红色箭头
        color = (0, 0, 255)
        direction = robot_angle
    else:
        # 后退：橙色箭头，反向
        color = (0, 165, 255)
        direction = robot_angle + 180°
else:  # 静止
    # 淡蓝色细箭头显示朝向
    color = (255, 200, 100)
```

## 优势

1. **直观性**：一眼看出机器人是前进、后退还是静止
2. **速度感知**：箭头长度随速度变化
3. **状态区分**：不同颜色代表不同状态
4. **调试友好**：便于观察机器人行为
